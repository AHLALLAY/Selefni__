# Conventions de Nommage - Salefni

## Composants React

Les composants React doivent utiliser la convention **PascalCase**.

### Exemples :
- `Header.jsx`
- `Footer.jsx`
- `SimulationForm.jsx`
- `CreditRequest.jsx`

## Fonctions et Variables

Les fonctions et variables doivent utiliser la convention **camelCase**.

### Exemples :
- `calculateMonthlyPayment()`
- `handleSubmit()`
- `userData`
- `isLoading`

## Fichiers et Dossiers

- **Composants** : PascalCase (ex: `Header.jsx`, `Footer.jsx`)
- **Pages** : camelCase (ex: `home.jsx`, `simulation.jsx`)
- **Services** : camelCase (ex: `api.js`, `authService.js`)
- **Hooks personnalisés** : camelCase avec préfixe `use` (ex: `useAuth.js`, `useCredit.js`)

## Constantes

Les constantes doivent utiliser **UPPER_SNAKE_CASE**.

### Exemples :
- `API_BASE_URL`
- `MAX_LOAN_AMOUNT`
- `DEFAULT_INTEREST_RATE`
